Инструкция по запуску программы тестового задания.
1.	Версия Java - Oracle Open JDK version 19.0.1

2. Убедитесь, что Java установлена и правильно настроена в переменной среды windows

3. Переместите файл Main.java в удобное для работы с ним место

4. Подготовьте набор тестовых данных (в теории программа может принять любое количество файлов с расширением .txt и исключить любые не соответствующие условию задания строки, единственное в чем я не уверен, что программа справится с данными большого объема, сортировка строк осуществляется по длине строки)

5. Условия запуска программы. Порядок ввода параметров может быть любой.
Для режима сортировки -d задает обратный порядок если он не указан то применяется прямой порядок
Для типа данных указать -i для обработки целых чисел  или -s для обработки строк 
Чтобы указать имя выходного файла, нужно ввести полный путь и имя файла, расширение файла указать как .out, файл будет сохранен в формате .txt, папки не создаются автоматически, путь уже должен существовать, конечная директория не должна содержать файлы.
В остальных параметрах указать через пробел путь к файлам txt
6. Пример запуска. Запустите командную строку Windows CMD с правами администратора.

7. В командной строке введите java и путь к нашей программе и условия запуска пример ниже    
java C:\Папкаспрограммой\Main.java -d -i "C:\вывод\file.out" "C:\test\text.txt" "C:\test\text1.txt" "C:\test\text2.txt" "C:\test\text3.txt" 

Запускаем нажатием клавиши Enter
